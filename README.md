# cykCpp
C++ implementation of CYK algorithm.

### Compilación:
Para compilar, ejecute el siguiente comando:
```
    g++ -std=c++11 -o cyk main.cpp
```
el cual, genera el binario "cyk", el cual contiene el programa compilado,
éste se ejecuta mediante:
```
    ./cyk
```
